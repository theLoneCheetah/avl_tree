## 1. Выбор проекта

Для выполнения работы в качестве модуля для тестирования был выбран учебный проект из курса алгоритмов и структур данных 1 курса. Проект реализует собственную модель AVL-дерева - сбалансированного двоичного дерева поиска, для которого верно, что для каждой вершины высота поддеревьев отличается не более чем на 1. Проект состоит из двух файлов:
* node.py - описывает класс узла дерева Node, хранящий ключевое значение value, дополнительную координату xcoord, ссылки на левое и правое поддерево и высоту поддеревьев
* avl_tree.py - содержит класс, реализующий методы для работы с AVL-деревом: функции вставки, удаления, выборки элементов.
Код исходного модуля расположен в папке bin/ проекта ([ссылка](https://github.com/theLoneCheetah/avl_tree/tree/main/bin)).
